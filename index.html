Základy JS - opakování

1: Co je JavaScript a k jakým účelům se používá?

Doplň vhodné odpovědi a využij k tomu nečíslovaného seznamu.

Použij v textu aspoň 3 internetové odkazy na zajímavé příklady využití JS!

  • JavaScript je multiplatformní, objektově orientovaný, událostmi řízený skriptovací jazyk.
  • je určený pro tvorbu moderních dynamických webů.
  • změna vzhledu i obsahu stránky na základě akcí uživatele

2: Jaká pravidla platí pro psaní kódu v JavaScriptu?

Napiš aspoň pět zásad ve podobě číslovaného seznamu.

Použij angličtinu!

  1. JavaScript is case sensitive
  2. Commands dont need to end with a semicolon only if there are more than one on one line
  3. Variables Must be defined before being used
  4. Strings have to be enclosed in quotation marks, either a single or double
  5. To enter comments in the script, use "//" to start a single line comment or the combination of "/*" and "*/" to enclose a multi-line comment.

3: Jak je možné propojit JavaScript s webovou stránkou?

Doplňte další možnosti ve formě kódu.

Opatřete kód poznámkami!

// 1. Vložit kód do tagu script
<script>console.log('Hello');</script>
// 2. Připojit externí soubor .js
<script src="demo.js"></script>
// 3. Vložit příkazy JS jako reakce na události (zastaralé)
<button onclick="alert('Ahoj');"></button>

4: Jak je možné v JavaScriptu deklarovat proměnné?

Uveďte na příkladech různé datové typy, které je možné využívat v JS.

Doplňte chybějící kód/text podle vzoru úvodních dvou řádků!

let x = 0; // s přiřazením hodnoty
let x; // bez přiřazení hodnoty
let x = null; // ...
let x = 5; // celé číslo (typ number)
let x = 5.55; // reálné číslo (typ number)
let x = 0.2e4; // zápis desetinného čísla pomocí exponentu
let x = "nevím"; // typ string
let prestupny = false // logický typ boolean

5: Jaké operátory můžeme používat v JavaScriptu?

Vypište operátory, které patří do uvedených skupin.

U méně obvyklých operátorů dopište do závorky jejich význam!

Matematické

+, -, *, /, % (modulo - zbytek po celočíselném dělení), ** (umocnění)

Přiřazovací

=, +=, -=, *=, /=, %= (vydělí proměnnou hodnotou pravého operandu a zbytek přiřadí proměnné), **=

Logické

&&, ||, !

Porovnávací

==, ===(stejná hodnota a stejný typ), !=, !==, >, <, >=, <=, ?

Bitové

&, |, ~(není), ^, <<, >>, >>>

Typové

typeof (typ proměnné), instanceof (testuje typ určitého předmětu)

6: Která popup okna lze využít v JavaScriptu?

K uvedeným příkladům připište do poznámky, co se stane na webové stránce.

Na konci doplňte aspoň dvě varování, proč bychom se měli vyhýbat použití popup oken v reálných webových aplikacích.

alert()

/* zobrazí se alert okno kde je napsané nezadal jsi správné heslo a pod touto hláškou bude tlačítko*/
alert('Nezadal jsi správné heslo!');

prompt()

/* Do proměnné password vloží heslo zadané uživatelem */
zobrazí se alert okno a tem se zobrazí pole na psaní kde uživatel napíše heslo které se pak vloží do proměnné password

confirm()

/* zobrazí se alert okno s otázkoui pokud chce uživatel přejít na seznam a pokud klikne na confirm stránka ho převeda na seznam.cz */
if (confirm('Chceš přejít na stránky Seznam.cz?')) location.href='https://seznam.cz';

Varování:Chybějící optimalizace pro mobilní zařízení, příliš časté zobrazování pop up oken může být otravné
